# 2026_Spiking_motifs_detection: Detection of spiking motifs of arbitrary length in neural activity using bounded synaptic delays

This repository contains the codes for the paper [Detection of spiking motifs of arbitrary length in neural activity using bounded synaptic delays](https://arxiv.org/abs/2511.15296).

## Description 

This repository contains two experiments in two separate folders:

- `/Random_patterns_experiments` contains experiment code on randomly generated spiking motifs.
- `/SHD_experiments` contains experiment code on the Spiking Heidelberg Digits (SHD) database.
